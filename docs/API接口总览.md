## 二、全局约定（摘要）

- **基础路径：** `/api/v1`
- **认证方式：** JWT Bearer Token（Access Token + Refresh Token）
- **响应格式：** `{ "code": 200, "message": "success", "data": {...} }`
- **分页参数：** `page`（默认1）、`page_size`（默认20，最大100）
- **ID格式：** 雪花算法 BIGINT，JSON中以字符串传输
- **时间格式：** ISO 8601（`2026-04-09T10:00:00Z`）
- **错误码：** 400参数错误 / 401未认证 / 403无权限 / 404不存在 / 409冲突 / 500服务端错误
