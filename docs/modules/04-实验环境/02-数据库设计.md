## 一、ER 关系概览

```
images (镜像主表)
  │
  ├── 1:N ── image_versions (镜像版本)
  │
  └── N:1 ── image_categories (镜像分类)

experiment_templates (实验模板主表)
  │
  ├── 1:N ── template_containers (模板容器配置)
  │             │
  │             └── N:1 ── image_versions (镜像版本)
  │
  ├── 1:N ── template_checkpoints (检查点定义)
  │
  ├── 1:N ── template_init_scripts (初始化脚本)
  │
  ├── 1:N ── template_sim_scenes (模板仿真场景配置)
  │             │
  │             └── N:1 ── sim_scenarios (仿真场景库)
  │
  ├── 1:N ── template_tags (模板标签关联)
  │             │
  │             └── N:1 ── tags (标签表)
  │
  ├── 1:N ── template_roles (多人实验角色定义)
  │
  ├── 1:N ── experiment_instances (实验实例)
  │             │
  │             ├── 1:N ── instance_containers (实例容器)
  │             │
  │             ├── 1:N ── checkpoint_results (检查点结果)
  │             │
  │             ├── 1:N ── instance_snapshots (实例快照)
  │             │
  │             └── 1:N ── instance_operation_logs (操作日志)
  │
  └── 1:N ── experiment_groups (实验分组)
               │
               ├── 1:N ── group_members (分组成员)
               │
               └── 1:N ── group_messages (组内消息)

sim_scenarios (仿真场景库)
  │
  └── N:N ── sim_link_groups (联动组定义) ── via sim_link_group_scenes

resource_quotas (资源配额)
```

## 二、表结构设计

### 2.1 image_categories — 镜像分类表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| name | VARCHAR(50) | NOT NULL | 分类名称 |
| code | VARCHAR(50) | NOT NULL, UNIQUE | 分类编码：chain_node / middleware / tool / base_env |
| description | VARCHAR(200) | NULL | 分类描述 |
| sort_order | INT | NOT NULL DEFAULT 0 | 排序序号 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**预设分类数据：**

| code | name |
|------|------|
| chain_node | 链节点镜像 |
| middleware | 中间件镜像 |
| tool | 工具镜像 |
| base_env | 环境基础镜像 |

---

### 2.2 images — 镜像主表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| category_id | BIGINT | NOT NULL, FK | 镜像分类ID |
| name | VARCHAR(100) | NOT NULL | 镜像名称（如 geth、fabric-peer） |
| display_name | VARCHAR(100) | NOT NULL | 显示名称（如 Go-Ethereum） |
| description | TEXT | NULL | 镜像描述 |
| icon_url | VARCHAR(500) | NULL | 镜像图标URL |
| ecosystem | VARCHAR(50) | NULL | 链生态：bitcoin / ethereum / fabric / chainmaker / fisco / solana / polkadot / cosmos / move / general |
| source_type | SMALLINT | NOT NULL DEFAULT 1 | 来源：1平台官方 2教师自定义 |
| uploaded_by | BIGINT | NULL | 上传者ID（教师自定义时） |
| school_id | BIGINT | NULL | 所属学校ID（教师自定义时） |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1正常 2待审核 3已下架 4审核拒绝 |
| review_comment | VARCHAR(500) | NULL | 审核意见 |
| reviewed_by | BIGINT | NULL | 审核人ID |
| reviewed_at | TIMESTAMP | NULL | 审核时间 |
| default_ports | JSONB | NULL | 默认端口映射：[{"port":8545,"protocol":"tcp","name":"RPC"}] |
| default_env_vars | JSONB | NULL | 默认环境变量：[{"key":"NETWORK_ID","value":"1337","desc":"网络ID"}] |
| default_volumes | JSONB | NULL | 默认挂载卷：[{"path":"/data","desc":"数据目录"}] |
| typical_companions | JSONB | NULL | 典型搭配：{"required":[{"image":"zookeeper","reason":"..."}],"recommended":[...],"optional":[...]} |
| required_dependencies | JSONB | NULL | 必要依赖镜像列表：["zookeeper","fabric-orderer"] |
| resource_recommendation | JSONB | NULL | 建议资源配置：{"cpu":"0.5","memory":"1Gi","disk":"2Gi"} |
| documentation_url | VARCHAR(500) | NULL | 镜像详细文档链接（如 /docs/images/geth） |
| usage_count | INT | NOT NULL DEFAULT 0 | 使用次数（冗余计数） |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |
| deleted_at | TIMESTAMP | NULL | 软删除时间 |

**索引：**

```sql
CREATE INDEX idx_images_category_id ON images(category_id);
CREATE INDEX idx_images_ecosystem ON images(ecosystem);
CREATE INDEX idx_images_source_type ON images(source_type);
CREATE INDEX idx_images_status ON images(status);
CREATE INDEX idx_images_school_id ON images(school_id);
CREATE INDEX idx_images_uploaded_by ON images(uploaded_by);
```

**状态枚举：**

```go
const (
    ImageStatusActive    = 1  // 正常
    ImageStatusPending   = 2  // 待审核
    ImageStatusOffline   = 3  // 已下架
    ImageStatusRejected  = 4  // 审核拒绝
)
```

### 2.3 image_versions — 镜像版本表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| image_id | BIGINT | NOT NULL, FK | 所属镜像ID |
| version | VARCHAR(50) | NOT NULL | 版本号（如 1.13、2.5、latest） |
| registry_url | VARCHAR(500) | NOT NULL | 镜像仓库地址（如 registry.lianjing.com/geth:1.13） |
| image_size | BIGINT | NULL | 镜像大小（字节） |
| digest | VARCHAR(200) | NULL | 镜像摘要（SHA256） |
| min_cpu | VARCHAR(20) | NULL | 最低CPU要求（如 0.5） |
| min_memory | VARCHAR(20) | NULL | 最低内存要求（如 512Mi） |
| min_disk | VARCHAR(20) | NULL | 最低磁盘要求（如 1Gi） |
| is_default | BOOLEAN | NOT NULL DEFAULT FALSE | 是否为默认版本 |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1正常 2已废弃 |
| scan_result | JSONB | NULL | 安全扫描结果 |
| scanned_at | TIMESTAMP | NULL | 最后扫描时间 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**

```sql
CREATE INDEX idx_image_versions_image_id ON image_versions(image_id);
CREATE UNIQUE INDEX uk_image_versions_image_version ON image_versions(image_id, version);
```

### 2.4 experiment_templates — 实验模板主表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| school_id | BIGINT | NOT NULL, FK | 所属学校ID |
| teacher_id | BIGINT | NOT NULL, FK | 创建教师ID |
| title | VARCHAR(200) | NOT NULL | 实验名称 |
| description | TEXT | NULL | 实验描述（Markdown） |
| objectives | TEXT | NULL | 实验目标 |
| instructions | TEXT | NULL | 实验说明/步骤引导（Markdown） |
| reference_materials | TEXT | NULL | 参考资料（Markdown） |
| experiment_type | SMALLINT | NOT NULL DEFAULT 2 | 实验类型：1纯仿真 2真实环境 3混合 |
| topology_mode | SMALLINT | NULL | 拓扑模式：1单人单节点 2单人多节点 3多人协作组网 4共享基础设施（真实环境/混合实验时必填） |
| judge_mode | SMALLINT | NOT NULL DEFAULT 1 | 判题模式：1纯自动 2纯手动 3混合 |
| auto_weight | DECIMAL(5,2) | NULL | 自动评分权重百分比（混合模式时） |
| manual_weight | DECIMAL(5,2) | NULL | 手动评分权重百分比（混合模式时） |
| total_score | DECIMAL(6,2) | NOT NULL DEFAULT 100 | 总分 |
| max_duration | INT | NULL | 最长运行时间（分钟） |
| idle_timeout | INT | NOT NULL DEFAULT 30 | 空闲超时时间（分钟） |
| cpu_limit | VARCHAR(20) | NULL | CPU限制（核数，真实环境/混合实验时） |
| memory_limit | VARCHAR(20) | NULL | 内存限制（真实环境/混合实验时） |
| disk_limit | VARCHAR(20) | NULL | 磁盘限制（真实环境/混合实验时） |
| score_strategy | SMALLINT | NOT NULL DEFAULT 1 | 成绩策略：1取最后一次 2取最高分 |
| is_shared | BOOLEAN | NOT NULL DEFAULT FALSE | 是否共享到平台实验库 |
| cloned_from_id | BIGINT | NULL | 克隆来源模板ID |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1草稿 2已发布 3已下架 |
| sim_layout | JSONB | NULL | 仿真场景布局配置：场景面板位置和大小 |
| k8s_config | JSONB | NULL | 生成的K8s编排配置（YAML转JSON存储，真实环境/混合实验时） |
| network_config | JSONB | NULL | 网络配置（端口映射、组网规则等） |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |
| deleted_at | TIMESTAMP | NULL | 软删除时间 |

**索引：**

```sql
CREATE INDEX idx_experiment_templates_school_id ON experiment_templates(school_id);
CREATE INDEX idx_experiment_templates_teacher_id ON experiment_templates(teacher_id);
CREATE INDEX idx_experiment_templates_experiment_type ON experiment_templates(experiment_type);
CREATE INDEX idx_experiment_templates_topology_mode ON experiment_templates(topology_mode);
CREATE INDEX idx_experiment_templates_status ON experiment_templates(status);
CREATE INDEX idx_experiment_templates_is_shared ON experiment_templates(is_shared) WHERE is_shared = TRUE;
```

**实验类型枚举：**

```go
const (
    ExperimentTypeSimulation = 1  // 纯仿真（SimEngine驱动，轻量场景算法容器）
    ExperimentTypeReal       = 2  // 真实环境（容器驱动）
    ExperimentTypeHybrid     = 3  // 混合（容器 + SimEngine联动）
)
```

**拓扑模式枚举：**

```go
const (
    TopologySingleNode  = 1  // 单人单节点
    TopologyMultiNode   = 2  // 单人多节点
    TopologyCollaborate = 3  // 多人协作组网
    TopologyShared      = 4  // 共享基础设施
)
```

**判题模式枚举：**

```go
const (
    JudgeModeAuto   = 1  // 纯自动
    JudgeModeManual = 2  // 纯手动
    JudgeModeHybrid = 3  // 混合
```

### 2.5 template_containers — 模板容器配置表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| template_id | BIGINT | NOT NULL, FK | 所属实验模板ID |
| image_version_id | BIGINT | NOT NULL, FK | 镜像版本ID |
| container_name | VARCHAR(100) | NOT NULL | 容器名称（如 geth-node-1） |
| deployment_scope | SMALLINT | NOT NULL DEFAULT 1 | 部署范围：1实例独享 2共享基础设施；仅 `topology_mode=4` 允许取 2 |
| role_id | BIGINT | NULL, FK | 关联角色ID（多人实验时，NULL表示所有角色共享） |
| env_vars | JSONB | NULL | 环境变量覆盖：[{"key":"xxx","value":"xxx"}] |
| ports | JSONB | NULL | 端口映射覆盖：[{"container":8545,"protocol":"tcp"}] |
| volumes | JSONB | NULL | 挂载卷覆盖：[{"host_path":"","container_path":"/data"}] |
| cpu_limit | VARCHAR(20) | NULL | 单容器CPU限制（覆盖模板级） |
| memory_limit | VARCHAR(20) | NULL | 单容器内存限制（覆盖模板级） |
| depends_on | JSONB | NULL | 依赖容器名称列表：["kafka","zookeeper"] |
| startup_order | INT | NOT NULL DEFAULT 0 | 启动顺序 |
| is_primary | BOOLEAN | NOT NULL DEFAULT FALSE | 是否为主容器（学生直接操作的容器） |
| sort_order | INT | NOT NULL DEFAULT 0 | 排序序号 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**

```sql
CREATE INDEX idx_template_containers_template_id ON template_containers(template_id);
CREATE INDEX idx_template_containers_image_version_id ON template_containers(image_version_id);
CREATE INDEX idx_template_containers_deployment_scope ON template_containers(deployment_scope);
```

### 2.6 template_checkpoints — 检查点定义表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| template_id | BIGINT | NOT NULL, FK | 所属实验模板ID |
| title | VARCHAR(200) | NOT NULL | 检查点名称 |
| description | TEXT | NULL | 检查点描述 |
| check_type | SMALLINT | NOT NULL | 类型：1后端脚本验证 2手动评分 3前端SimEngine状态断言 |
| script_content | TEXT | NULL | 检查脚本内容（后端脚本类型时） |
| script_language | VARCHAR(20) | NULL | 脚本语言：bash / python |
| target_container | VARCHAR(100) | NULL | 目标容器名称（脚本在哪个容器执行，后端脚本类型时） |
| assertion_config | JSONB | NULL | SimEngine状态断言配置（前端断言类型时） |
| score | DECIMAL(6,2) | NOT NULL | 该检查点分值 |
| scope | SMALLINT | NOT NULL DEFAULT 1 | 范围：1个人 2组级（多人实验时） |
| sort_order | INT | NOT NULL DEFAULT 0 | 排序序号（即步骤顺序） |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**

```sql
CREATE INDEX idx_template_checkpoints_template_id ON template_checkpoints(template_id);
```

**检查类型枚举：**

```go
const (
    CheckTypeScript    = 1  // 后端脚本验证（进入容器执行）

)
```

### 2.8 sim_scenarios — 仿真场景库表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |

| icon_url | VARCHAR(500) | NULL | 场景图标URL |
| thumbnail_url | VARCHAR(500) | NULL | 场景缩略图URL |
| source_type | SMALLINT | NOT NULL DEFAULT 1 | 来源：1平台内置 2教师自定义 |
| uploaded_by | BIGINT | NULL | 上传者ID（教师自定义时） |
| school_id | BIGINT | NULL | 所属学校ID（教师自定义时） |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1正常 2待审核 3已下架 4审核拒绝 |
| review_comment | VARCHAR(500) | NULL | 审核意见 |
| reviewed_by | BIGINT | NULL | 审核人ID |
| reviewed_at | TIMESTAMP | NULL | 审核时间 |
| algorithm_type | VARCHAR(100) | NOT NULL | 算法类型标识（如 pbft-consensus / sha256-hash / ecdsa-sign / pow-mining） |
| time_control_mode | VARCHAR(20) | NOT NULL DEFAULT 'process' | 时间控制模式：process（过程化）/ reactive（交互响应式）/ continuous（持续运行式） |
| container_image_url | VARCHAR(500) | NULL | 场景算法容器镜像地址（平台内置场景预设，教师自定义场景上传后填入） |
| container_image_size | BIGINT | NULL | 场景算法容器镜像大小（字节，通常10-50MB） |
| default_params | JSONB | NULL | 场景默认参数（如节点数、难度、延迟等） |
| interaction_schema | JSONB | NULL | 场景支持的交互操作定义（操作面板配置） |
| data_source_mode | SMALLINT | NOT NULL DEFAULT 1 | 数据源模式：1仿真模式（场景算法容器驱动）2采集模式（Collector Agent从实验容器采集）3双模式 |
| default_size | JSONB | NULL | 默认尺寸：{"width":6,"height":4}（栅格单位） |
| delivery_phase | SMALLINT | NOT NULL DEFAULT 1 | 交付阶段：1/2/3 |
| version | VARCHAR(50) | NOT NULL DEFAULT '1.0.0' | 场景版本 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |
| deleted_at | TIMESTAMP | NULL | 软删除时间 |

**索引：**

```sql
CREATE INDEX idx_sim_scenarios_category ON sim_scenarios(category);
CREATE INDEX idx_sim_scenarios_source_type ON sim_scenarios(source_type);
CREATE INDEX idx_sim_scenarios_status ON sim_scenarios(status);
CREATE INDEX idx_sim_scenarios_algorithm_type ON sim_scenarios(algorithm_type);
CREATE INDEX idx_sim_scenarios_delivery_phase ON sim_scenarios(delivery_phase);
```

### 2.9 sim_link_groups — 联动组定义表

| 字段 | 类型 | 约束 | 说明 |

## 三、Redis 数据结构

### 3.1 实验实例状态缓存

```
# 实验实例实时状态（高频读取）
Key:   exp:instance:{instance_id}:status
Value: { "status": 3, "experiment_type": 1, "started_at": "2026-04-08T10:00:00Z", "last_active_at": "2026-04-08T10:25:00Z" }
TTL:   无过期（实例销毁时删除）
```

### 3.2 学生并发实验计数

```
# 学生当前运行中的实验数量（仅计算真实环境和混合实验，纯仿真不计入）
Key:   exp:student:{student_id}:running_count
Value: 整数（当前运行中的实验数）
TTL:   无过期（实验启动+1，停止-1）
```

### 3.3 课程并发实验计数

```
# 课程当前并发实验数（仅计算真实环境和混合实验）
Key:   exp:course:{course_id}:concurrency
Value: 整数
TTL:   无过期（实验启动+1，停止-1）
```

### 3.4 实验启动排队

```
# 资源不足时的排队队列
Key:   exp:queue:{course_id}
Value: List，元素为 { "student_id": xxx, "template_id": xxx, "requested_at": "..." }
TTL:   无过期（队列消费后移除）
```

### 3.5 实例活跃心跳

```
# 实例最后活跃时间（用于空闲超时检测）
Key:   exp:heartbeat:{instance_id}
Value: 最后活跃时间戳
TTL:   35分钟（超过30分钟无更新即视为空闲）
```

### 3.6 教师监控面板缓存

```
# 课程下所有实验实例的状态汇总（教师监控用）
Key:   exp:monitor:{course_id}:{template_id}
Value: Hash，field为student_id，value为 { "status": 3, "experiment_type": 2, "checkpoints_passed": 3, "checkpoints_total": 5 }
TTL:   30秒（定时刷新）
```

## 四、定时任务

| 任务 | 执行频率 | 说明 |
|------|---------|------|
| 空闲超时检测 | 每5分钟 | 检查运行中实例的最后活跃时间，超过空闲超时阈值的自动保存快照并回收 |
| 最长运行时间检测 | 每5分钟 | 检查运行中实例是否超过模板设置的最长运行时间，超时则强制快照并回收 |
| 自动快照 | 每5分钟 | 对运行中的实例自动保存增量快照（真实环境：容器快照；纯仿真：SimEngine状态快照） |
| 资源使用量同步 | 每1分钟 | 从K8s API同步各实例的实际CPU/内存使用量到数据库和Redis |
| 排队请求处理 | 每30秒 | 检查排队队列，当资源释放时自动创建排队中的实验实例 |
| 过期快照清理 | 每天凌晨3:00 | 清理已销毁实例的过期快照（保留最近30天） |
| 课程结束环境回收 | 每小时 | 检查已结束课程关联的运行中实验环境，强制回收 |
