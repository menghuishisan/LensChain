## 三、全局交互规范

### 3.1 学校状态对用户体验的影响

| 学校状态 | 校管体验 | 教师/学生体验 |
|---------|---------|-------------|
| 已激活 | 正常使用所有功能 | 正常使用 |
| 缓冲期 | 顶部显示红色倒计时横幅，功能正常 | 正常使用，不感知 |
| 已冻结 | 登录后仅显示冻结提示页 | 无法登录，提示"学校授权已过期" |
| 已注销 | 无法登录 | 无法登录 |

### 3.2 登录时的学校状态校验

登录接口需增加学校状态校验（在模块01的登录流程中集成）：

1. 查询用户所属学校状态
2. 学校为冻结/注销状态 → 拒绝登录，返回对应提示
3. 学校为缓冲期 → 允许登录，前端显示提醒
