## 三、全局交互规范

### 3.1 视频播放进度上报

- 前端每30秒自动上报一次播放进度
- 页面离开时立即上报当前进度
- 视频播放到95%以上视为完成该课时

### 3.2 作业自动保存

- 学生作答过程中每60秒自动保存草稿
- 页面离开时提示"有未保存的作答，是否离开？"
- 草稿保存在localStorage + 服务端双重保障

### 3.3 课程状态对页面的影响

| 课程状态 | 教师可操作 | 学生可操作 |
|---------|-----------|-----------|
| 草稿 | 编辑所有内容 | 不可见 |
| 已发布 | 编辑内容、管理学生 | 可加入、浏览内容 |
| 进行中 | 编辑内容、批改作业 | 学习、提交作业 |
| 已结束 | 批改作业、查看数据 | 查看内容和成绩，不可提交 |
| 已归档 | 查看和导出数据 | 不可见 |
