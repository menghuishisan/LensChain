## 一、ER 关系概览

```
courses (课程主表)
  │
  ├── 1:N ── chapters (章节)
  │             │
  │             └── 1:N ── lessons (课时)
  │                          │
  │                          ├── 1:N ── lesson_attachments (课时附件)
  │                          └── N:1 ── experiments (实验模板，定义在模块04)
  │
  ├── 1:N ── course_enrollments (选课记录)
  │
  ├── 1:N ── course_experiments (课程独立实验关联)
  │
  ├── 1:N ── assignments (作业/测验)
  │             │
  │             ├── 1:N ── assignment_questions (题目)
  │             │
  │             ├── 1:1 ── assignment_drafts (学生当前草稿，按 assignment_id + student_id 唯一)
  │             │
  │             └── 1:N ── assignment_submissions (学生提交)
  │                          │
  │                          └── 1:N ── submission_answers (答题记录)
  │
  ├── 1:N ── course_announcements (课程公告)
  │
  ├── 1:N ── course_discussions (讨论帖)
  │             │
  │             ├── 1:N ── discussion_replies (回复)
  │             └── 1:N ── discussion_likes (点赞)
  │
  ├── 1:N ── course_evaluations (课程评价)
  │
  ├── 1:N ── learning_progresses (学习进度)
  │
  ├── 1:N ── course_schedules (课程表)
  │
  ├── 1:1 ── course_grade_configs (成绩权重配置)
  │
  └── 1:N ── course_grade_overrides (课程成绩调整记录)
```

## 二、表结构设计

### 2.1 courses — 课程主表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| school_id | BIGINT | NOT NULL, FK | 所属学校ID |
| teacher_id | BIGINT | NOT NULL, FK | 负责教师ID |
| title | VARCHAR(200) | NOT NULL | 课程名称 |
| description | TEXT | NULL | 课程描述 |
| cover_url | VARCHAR(500) | NULL | 封面图URL |
| course_type | SMALLINT | NOT NULL | 类型：1理论 2实验 3混合 4项目实战 |
| difficulty | SMALLINT | NOT NULL DEFAULT 1 | 难度：1入门 2进阶 3高级 4研究 |
| topic | VARCHAR(50) | NOT NULL | 主题分类 |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1草稿 2已发布 3进行中 4已结束 5已归档 |
| is_shared | BOOLEAN | NOT NULL DEFAULT FALSE | 是否共享到课程库 |
| invite_code | VARCHAR(10) | NULL | 邀请码（6位字母数字） |
| start_at | TIMESTAMP | NULL | 课程开始时间 |
| end_at | TIMESTAMP | NULL | 课程结束时间 |
| credits | DECIMAL(3,1) | NULL | 课程学分（如 3.0、1.5），供模块06计算GPA使用 |
| semester_id | BIGINT | NULL | 所属学期ID（引用模块06 `semesters.id`，供成绩审核关联） |
| max_students | INT | NULL | 最大学生数（NULL不限制） |
| cloned_from_id | BIGINT | NULL | 克隆来源课程ID（NULL为原创） |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |
| deleted_at | TIMESTAMP | NULL | 软删除时间 |

**索引：**

```sql
CREATE INDEX idx_courses_school_id ON courses(school_id);
CREATE INDEX idx_courses_teacher_id ON courses(teacher_id);
CREATE INDEX idx_courses_status ON courses(status);
CREATE INDEX idx_courses_course_type ON courses(course_type);
CREATE INDEX idx_courses_is_shared ON courses(is_shared) WHERE is_shared = TRUE;
CREATE UNIQUE INDEX uk_courses_invite_code ON courses(invite_code) WHERE invite_code IS NOT NULL AND deleted_at IS NULL;
CREATE INDEX idx_courses_start_at ON courses(start_at);
CREATE INDEX idx_courses_end_at ON courses(end_at);
CREATE INDEX idx_courses_semester_id ON courses(semester_id);
```

**状态枚举：**

```go
const (
    CourseStatusDraft     = 1  // 草稿
    CourseStatusPublished = 2  // 已发布
    CourseStatusActive    = 3  // 进行中
    CourseStatusEnded     = 4  // 已结束
    CourseStatusArchived  = 5  // 已归档
)
```

### 2.9 assignment_drafts — 作答草稿表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| assignment_id | BIGINT | NOT NULL, FK | 作业ID |
| student_id | BIGINT | NOT NULL, FK | 学生ID |
| answers | JSONB | NOT NULL DEFAULT '[]'::jsonb | 当前草稿答案列表 |
| saved_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 最近保存时间 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**

```sql
CREATE UNIQUE INDEX uk_assignment_drafts_assignment_student ON assignment_drafts(assignment_id, student_id);
CREATE INDEX idx_assignment_drafts_student_id ON assignment_drafts(student_id);
CREATE INDEX idx_assignment_drafts_saved_at ON assignment_drafts(saved_at);
```

**设计说明：**

- `answers` 以 JSONB 保存当前作答快照，结构与提交接口中的 `answers` 数组保持一致
- 每个学生在每个作业下仅保留一份最新草稿，重复保存时覆盖更新
- 正式提交后，服务端草稿立即删除或标记失效；当前版本采用“提交后删除草稿”策略
- 草稿不参与提交次数、成绩、批改、统计等正式提交流程
