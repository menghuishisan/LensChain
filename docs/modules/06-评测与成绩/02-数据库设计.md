# 评测与成绩模块 — 数据库设计

> 模块状态：✅ 已确认
> 文档版本：v1.0

---

## 一、ER 关系总览

```
                    ┌─────────────────────────────────────────────┐
                    │           外部模块（只读引用）                 │
                    │  users (模块01)                              │
                    │  schools (模块02)                            │
                    │  courses / course_grade_configs /            │
                    │  course_enrollments / assignment_submissions │
                    │  (模块03)                                    │
                    └─────────────────────────────────────────────┘
                                        │ 读取
                                        ▼
semesters (学期) ──1:N──→ grade_reviews (成绩审核)
     │                         │
     │                    关联 courses
     │
     └──1:N──→ student_semester_grades (学期成绩汇总)
                    │
                    └──→ grade_appeals (成绩申诉)

grade_level_configs (等级映射配置) ── 按 school_id 隔离

academic_warnings (学业预警) ── 关联 student + semester

transcript_records (成绩单生成记录)
```

## 二、表结构设计

### 2.1 semesters — 学期表

> 定义学校的学期信息，用于成绩按学期组织。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| school_id | BIGINT | NOT NULL | 所属学校ID（引用 `schools.id`） |
| name | VARCHAR(50) | NOT NULL | 学期名称（如"2025-2026学年第一学期"） |
| code | VARCHAR(20) | NOT NULL | 学期编码（如"2025-2026-1"） |
| start_date | DATE | NOT NULL | 学期开始日期 |
| end_date | DATE | NOT NULL | 学期结束日期 |
| is_current | BOOLEAN | NOT NULL DEFAULT FALSE | 是否为当前学期 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |
| deleted_at | TIMESTAMP | NULL | 软删除 |

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| idx_semesters_school_id | school_id | 普通 | 按学校查询 |
| uk_semesters_school_code | (school_id, code) | 唯一（partial） | 同校学期编码唯一，WHERE deleted_at IS NULL |
| idx_semesters_is_current | (school_id, is_current) | 普通 | 查询当前学期 |

---

### 2.2 grade_level_configs — 等级映射配置表

> 每个学校可自定义百分制→等级→绩点的映射规则。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| school_id | BIGINT | NOT NULL | 所属学校ID |
| level_name | VARCHAR(10) | NOT NULL | 等级名称（如 A、B+、B、C、D、F） |
| min_score | DECIMAL(5,2) | NOT NULL | 最低分（含） |
| max_score | DECIMAL(5,2) | NOT NULL | 最高分（含） |
| gpa_point | DECIMAL(3,2) | NOT NULL | 对应绩点值（0.00-4.00） |
| sort_order | INT | NOT NULL DEFAULT 0 | 排序（分数从高到低） |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| idx_grade_level_configs_school_id | school_id | 普通 | 按学校查询 |
| uk_grade_level_configs_school_level | (school_id, level_name) | 唯一 | 同校等级名唯一 |

**默认数据（系统初始化时为每个学校插入）：**

| level_name | min_score | max_score | gpa_point | sort_order |
|-----------|-----------|-----------|-----------|------------|
| A | 90.00 | 100.00 | 4.00 | 1 |
| B | 80.00 | 89.99 | 3.00 | 2 |
| C | 70.00 | 79.99 | 2.00 | 3 |
| D | 60.00 | 69.99 | 1.00 | 4 |
| F | 0.00 | 59.99 | 0.00 | 5 |

---

### 2.3 grade_reviews — 成绩审核表

> 记录教师提交成绩审核和学校管理员审核的流程。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| course_id | BIGINT | NOT NULL | 课程ID（引用 `courses.id`） |
| school_id | BIGINT | NOT NULL | 学校ID |
| semester_id | BIGINT | NOT NULL | 学期ID（引用 `semesters.id`） |
| submitted_by | BIGINT | NOT NULL | 提交教师ID（引用 `users.id`） |
| status | SMALLINT | NOT NULL DEFAULT 1 | 审核状态：1未提交 2待审核 3已通过 4已驳回 |
| submit_note | TEXT | NULL | 教师提交备注 |
| submitted_at | TIMESTAMP | NULL | 提交时间 |
| reviewed_by | BIGINT | NULL | 审核人ID |
| reviewed_at | TIMESTAMP | NULL | 审核时间 |
| review_comment | TEXT | NULL | 审核意见（通过/驳回均可填写） |
| is_locked | BOOLEAN | NOT NULL DEFAULT FALSE | 成绩是否已锁定 |
| locked_at | TIMESTAMP | NULL | 锁定时间 |
| unlocked_by | BIGINT | NULL | 解锁操作人ID |
| unlocked_at | TIMESTAMP | NULL | 解锁时间 |
| unlock_reason | TEXT | NULL | 解锁原因 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| idx_grade_reviews_school_id | school_id | 普通 | 按学校查询 |
| idx_grade_reviews_semester_id | semester_id | 普通 | 按学期查询 |
| idx_grade_reviews_status | status | 普通 | 按审核状态筛选 |
| uk_grade_reviews_course_semester | (course_id, semester_id) | 唯一 | 每门课程每学期只有一条审核记录 |
| idx_grade_reviews_submitted_by | submitted_by | 普通 | 教师查询自己提交的审核 |

**状态枚举：**
| 值 | 常量名 | 说明 |
|----|--------|------|
| 1 | ReviewStatusNotSubmitted | 未提交 |
| 2 | ReviewStatusPending | 待审核 |
| 3 | ReviewStatusApproved | 已通过 |
| 4 | ReviewStatusRejected | 已驳回 |

---

### 2.4 student_semester_grades — 学生学期成绩汇总表

> 存储学生每学期每门课程的最终成绩、等级和绩点。审核通过后由系统自动生成/更新。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| student_id | BIGINT | NOT NULL | 学生ID（引用 `users.id`） |
| school_id | BIGINT | NOT NULL | 学校ID |
| semester_id | BIGINT | NOT NULL | 学期ID |
| course_id | BIGINT | NOT NULL | 课程ID |
| final_score | DECIMAL(6,2) | NOT NULL | 最终百分制成绩（来自模块03加权计算结果） |
| grade_level | VARCHAR(10) | NOT NULL | 等级（A/B/C/D/F，根据等级映射配置计算） |
| gpa_point | DECIMAL(3,2) | NOT NULL | 绩点值 |
| credits | DECIMAL(3,1) | NOT NULL | 课程学分 |
| is_adjusted | BOOLEAN | NOT NULL DEFAULT FALSE | 是否经过手动调整（模块03中教师调整） |
| review_id | BIGINT | NOT NULL | 关联审核记录ID |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| uk_student_semester_grades_unique | (student_id, semester_id, course_id) | 唯一 | 每人每学期每课程一条记录 |
| idx_ssg_school_id | school_id | 普通 | 按学校查询 |
| idx_ssg_semester_id | semester_id | 普通 | 按学期查询 |
| idx_ssg_student_id | student_id | 普通 | 按学生查询 |
| idx_ssg_course_id | course_id | 普通 | 按课程查询 |
| idx_ssg_grade_level | grade_level | 普通 | 按等级筛选 |

---

### 2.5 grade_appeals — 成绩申诉表

> 学生对已审核通过的课程成绩提出申诉。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| student_id | BIGINT | NOT NULL | 申诉学生ID |
| school_id | BIGINT | NOT NULL | 学校ID |
| semester_id | BIGINT | NOT NULL | 学期ID |
| course_id | BIGINT | NOT NULL | 课程ID |
| grade_id | BIGINT | NOT NULL | 关联的成绩记录ID（`student_semester_grades.id`） |
| original_score | DECIMAL(6,2) | NOT NULL | 申诉时的原始成绩 |
| appeal_reason | TEXT | NOT NULL | 申诉理由（至少20字） |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1待处理 2已同意 3已驳回 |
| handled_by | BIGINT | NULL | 处理教师ID |
| handled_at | TIMESTAMP | NULL | 处理时间 |
| new_score | DECIMAL(6,2) | NULL | 修改后的新成绩（仅同意时） |
| handle_comment | TEXT | NULL | 处理意见 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| idx_grade_appeals_student_id | student_id | 普通 | 学生查询自己的申诉 |
| idx_grade_appeals_course_id | course_id | 普通 | 按课程查询 |
| idx_grade_appeals_status | status | 普通 | 按状态筛选 |
| uk_grade_appeals_student_course_semester | (student_id, course_id, semester_id) | 唯一 | 每人每课程每学期最多1次申诉 |
| idx_grade_appeals_school_id | school_id | 普通 | 按学校查询 |

**状态枚举：**
| 值 | 常量名 | 说明 |
|----|--------|------|
| 1 | AppealStatusPending | 待处理 |
| 2 | AppealStatusApproved | 已同意 |
| 3 | AppealStatusRejected | 已驳回 |

### 2.6 academic_warnings — 学业预警表

> 系统自动检测并生成的学业预警记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| student_id | BIGINT | NOT NULL | 学生ID |
| school_id | BIGINT | NOT NULL | 学校ID |
| semester_id | BIGINT | NOT NULL | 触发学期ID |
| warning_type | SMALLINT | NOT NULL | 预警类型：1低GPA 2连续挂科 |
| detail | JSONB | NOT NULL | 预警详情 |
| status | SMALLINT | NOT NULL DEFAULT 1 | 状态：1待处理 2已处理 3已解除 |
| handled_by | BIGINT | NULL | 处理人ID |
| handled_at | TIMESTAMP | NULL | 处理时间 |
| handle_note | TEXT | NULL | 处理备注 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

**`detail` JSONB 结构：**

低GPA预警：
```json
{
  "current_gpa": 1.85,
  "threshold": 2.0,
  "semester_courses": [
    { "course_id": "...", "course_name": "区块链原理", "score": 55, "grade": "F", "credits": 3.0 }
  ]
}
```

连续挂科预警：
```json
{
  "fail_count": 3,
  "threshold": 2,
  "failed_courses": [
    { "course_id": "...", "course_name": "智能合约开发", "score": 48, "semester": "2025-2026-1" },
    { "course_id": "...", "course_name": "密码学基础", "score": 52, "semester": "2025-2026-1" }
  ]
}
```

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| idx_academic_warnings_student_id | student_id | 普通 | 按学生查询 |
| idx_academic_warnings_school_id | school_id | 普通 | 按学校查询 |
| idx_academic_warnings_semester_id | semester_id | 普通 | 按学期查询 |
| idx_academic_warnings_status | status | 普通 | 按状态筛选 |

**状态枚举：**
| 值 | 常量名 | 说明 |
|----|--------|------|
| 1 | WarningStatusPending | 待处理 |
| 2 | WarningStatusHandled | 已处理 |
| 3 | WarningStatusResolved | 已解除 |

**预警类型枚举：**
| 值 | 常量名 | 说明 |
|----|--------|------|
| 1 | WarningTypeLowGPA | 低GPA预警 |
| 2 | WarningTypeConsecutiveFail | 连续挂科预警 |

### 2.7 warning_configs — 学业预警配置表

> 每个学校的学业预警阈值配置。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| school_id | BIGINT | NOT NULL, UNIQUE | 学校ID（每校一条配置） |
| gpa_threshold | DECIMAL(3,2) | NOT NULL DEFAULT 2.00 | GPA预警阈值 |
| fail_count_threshold | INT | NOT NULL DEFAULT 2 | 连续挂科预警门数 |
| is_enabled | BOOLEAN | NOT NULL DEFAULT TRUE | 是否启用预警 |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |
| updated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 更新时间 |

---

### 2.8 transcript_records — 成绩单生成记录表

> 记录成绩单PDF的生成历史。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | 雪花算法ID |
| school_id | BIGINT | NOT NULL | 学校ID |
| student_id | BIGINT | NOT NULL | 学生ID |
| generated_by | BIGINT | NOT NULL | 生成操作人ID |
| file_url | VARCHAR(500) | NOT NULL | PDF文件URL（MinIO存储） |
| file_size | BIGINT | NOT NULL | 文件大小（字节） |
| include_semesters | JSONB | NOT NULL | 包含的学期列表 |
| generated_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 生成时间 |
| expires_at | TIMESTAMP | NULL | 文件过期时间（可选，过期后自动清理） |
| created_at | TIMESTAMP | NOT NULL DEFAULT NOW() | 创建时间 |

**`include_semesters` JSONB 结构：**
```json
["2024-2025-1", "2024-2025-2", "2025-2026-1"]
```

**索引：**
| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| idx_transcript_records_student_id | student_id | 普通 | 按学生查询 |
| idx_transcript_records_school_id | school_id | 普通 | 按学校查询 |

## 三、与外部模块的数据依赖

### 3.1 读取的外部表（只读，不修改）

| 外部表 | 所属模块 | 读取字段 | 用途 |
|--------|---------|---------|------|
| `users` | 模块01 | id, name, student_no, school_id | 学生信息展示 |
| `schools` | 模块02 | id, name, logo_url | 学校信息、成绩单Logo |
| `courses` | 模块03 | id, title, school_id, teacher_id, credits*, semester*, status | 课程信息、学分、学期 |
| `course_grade_configs` | 模块03 | course_id, config | 成绩权重配置，用于展示成绩构成 |
| `course_enrollments` | 模块03 | course_id, student_id | 选课关系 |
| `assignment_submissions` | 模块03 | assignment_id, student_id, score_after_deduction | 各项作业最终得分 |

> *标注 `*` 的字段为模块03需要新增的字段：
> - `courses.credits DECIMAL(3,1) NULL` — 课程学分
> - `courses.semester_id BIGINT NULL` — 关联学期ID（引用 `semesters.id`）

### 3.2 写入的外部表

| 外部表 | 所属模块 | 写入场景 |
|--------|---------|---------|
| `operation_logs` | 模块01 | 成绩审核通过/驳回、申诉处理、预警处理等操作记录 |

### 3.3 通过接口调用的外部模块

| 外部模块 | 调用接口 | 场景 |
|---------|---------|------|
| 模块07 通知与消息 | 发送通知接口 | 审核通过/驳回通知教师、申诉处理通知学生、学业预警通知学生 |

## 四、Redis 缓存设计

| Key 格式 | 类型 | TTL | 说明 |
|---------|------|-----|------|
| `gpa:student:{student_id}` | Hash | 24h | 学生GPA缓存（semester_gpa, cumulative_gpa） |
| `grade_level_config:{school_id}` | String(JSON) | 1h | 学校等级映射配置缓存 |
| `semester:current:{school_id}` | String | 1h | 当前学期ID缓存 |
| `warning_config:{school_id}` | String(JSON) | 1h | 预警配置缓存 |

## 五、定时任务

| 任务 | 执行频率 | 说明 |
|------|---------|------|
| 学业预警检测 | 每次成绩审核通过后触发 | 检查该学期该学校所有学生的GPA和挂科情况 |
| GPA缓存刷新 | 每日凌晨3:00 | 全量刷新所有学生的GPA缓存 |
| 成绩单文件清理 | 每日凌晨4:00 | 清理已过期的成绩单PDF文件 |
| 预警自动解除检测 | 每次成绩更新后触发 | 检查已预警学生的GPA是否回升到阈值以上 |
