## 三、全局交互规范

### 3.1 自动刷新

- 运维仪表盘：每30秒自动刷新
- 告警事件列表：每60秒自动刷新
- 其他页面：手动刷新

### 3.2 告警级别颜色

| 级别 | 颜色 | 图标 |
|------|------|------|
| 信息 | 蓝色 | 🔵 |
| 警告 | 黄色 | 🟡 |
| 严重 | 红色 | 🔴 |
| 紧急 | 深红色 | 🔴（闪烁） |

### 3.3 导航位置

模块08的所有页面位于超级管理员侧边栏的"系统管理"分组下：
```
系统管理
├── 运维仪表盘
├── 统一审计
├── 全局配置
├── 告警管理
│   ├── 告警规则
│   └── 告警事件
├── 平台统计
└── 数据备份
```
