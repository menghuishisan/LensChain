## 二、全局约定（摘要）

- **主键：** 雪花算法 BIGINT
- **软删除：** `deleted_at TIMESTAMP NULL`
- **时间字段：** `created_at`、`updated_at`（TIMESTAMP NOT NULL DEFAULT NOW()）
- **命名规范：** 表名蛇形复数（`users`）、字段蛇形单数（`user_id`）
- **多租户：** 业务表含 `school_id` 字段，查询时隔离
- **枚举值：** SMALLINT 存储，代码中定义常量

## 四、Redis 缓存总览

### 模块01 — 用户与认证（5个Key）

| Key 模式 | 类型 | TTL | 说明 |
|----------|------|-----|------|
| `session:{user_id}` | String | 7天 | 用户Refresh Token |
| `token_blacklist:{jti}` | String | Token剩余有效期 | Access Token黑名单 |
| `login_fail:{phone}` | String | 30分钟 | 登录失败计数 |
| `account_locked:{phone}` | String | 锁定时长 | 账号锁定标记 |
| `config:security_policy` | String | 永久 | 安全策略配置 |

### 模块02 — 学校与租户管理（2个Key）

| Key 模式 | 类型 | TTL | 说明 |
|----------|------|-----|------|

| `school:expiring_soon` | Sorted Set | 永久 | 即将到期学校集合 |

### 模块04 — 实验环境（8个Key）

| `ctf:score:{comp_id}:{challenge_id}` | JSON | 永久 | 动态分值缓存 |
| `ctf:ad:round:{comp_id}:{group_id}` | JSON | 永久 | 当前回合状态 |

| `ctf:ad:exploit:{comp}:{group}:{challenge}` | Integer | 永久 | 漏洞利用计数 |
